\### Architectural Timeline Summary:



\* \*\*GCN (Graphics Core Next):\*\* The R9 290X through Radeon VII used the GCN architecture. This era saw AMD experiment with \*\*HBM (High-Bandwidth Memory)\*\*, which allowed for unprecedented memory bandwidth in the Fury X and Radeon VII, despite using a smaller memory bus width.

\* \*\*RDNA (Radeon DNA):\*\* The introduction of \*\*RDNA 1\*\* (RX 5000 series, not in table) and \*\*RDNA 2\*\* (RX 6900 XT) marked a major pivot, heavily optimizing the GPU for gaming performance and power efficiency. \*\*Ray Tracing\*\* hardware was introduced here.

\* \*\*RDNA 3:\*\* The latest generation (RX 7900 XTX) is AMD's first \*\*chiplet-based\*\* GPU, which splits the graphics processing and memory into separate dies, leading to massive increases in core count and performance.
